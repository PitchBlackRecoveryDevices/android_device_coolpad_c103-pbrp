# android_device_coolpad_c103
For building TWRP for coolpad cool 1 dual

To compile

. build/envsetup.sh && lunch omni_c103-eng && make -j8 recoveryimage


## Device specifications

| Device                  | Coolpad cool 1 dual                                         |
| ----------------------- | :---------------------------------------------------------- |
| SoC                     | Snapdragon 652                                              |
| GPU                     | Adreno 510                                                  |
| Memory                  | 3GB RAM                                                     |
